## INSTALLATION

clone the repo

```
https://github.com/matfaisall/react-js-test.git
```

install the dependencies

```
yarn add
```

start the server

```
yarn server
```

start the development server

```
yarn dev
```

## BUILD WITH :

1. React Vite
2. Typescript
3. Material Ui
4. Axios
5. React Router Dom
